Why This Only Works on Rialo

The resolution engine of RialoPredict is built entirely on two Rialo primitives unavailable anywhere else. Here is what we are doing in production code:

1. Native HTTP Oracle — no middleware

// Contract calls external APIs directly. No Chainlink. No keeper bot.
#[rialo::async_handler(event = "schedule_resolve")]
pub async fn resolve_market(ctx: Context<ResolveMarket>) -> Result<()> {
    let response = HttpClient::fetch(
        HttpRequest::get(&market.resolution_url).build()
    ).await?;

    let data: serde_json::Value = serde_json::from_slice(&response.body)?;
    let winner = determine_winner(&market.outcomes, &data)?;
    market.winning_outcome = Some(winner);
    market.resolved = true;
    // Payouts distributed automatically to all winners
    Ok(())
}

2. Async Event Scheduling — no cron job

// At market creation, Rialo schedules the resolution wakeup.
// The contract wakes itself up at close_timestamp. Zero external infra.
emit_event("schedule_resolve", ScheduleResolvePayload {
    market:     market.key(),
    resolve_at: params.close_timestamp,
})?;

On Ethereum or Solana, the same flow requires: a Chainlink oracle subscription, a Gelato/Keep3r keeper bot running 24/7, a manual admin resolution transaction, and a separate claim transaction per winner.

On Rialo: one contract, one deploy. Everything runs itself.

Why This Matters for Africa

Existing on-chain prediction markets (Polymarket, Augur, Manifold) are dominated by USD-denominated crypto events targeting Western audiences. RialoPredict is built specifically for the Nigerian and broader African market:

  • USD/NGN 1-minute markets — the most traded currency pair in West Africa
  • 2027 Nigerian Presidential Election — the largest democracy on the continent
  • African World Cup semifinalist market — 2026 WC is the first to include 48 teams, Africa gets 9 slots
  • Victor Osimhen in the top-scorer market — Nigeria's biggest football star
  • Naira-denominated display (₦) throughout the UI
  • Email login removes seed-phrase barrier for crypto-new users

This is a genuinely new audience for Web3 prediction markets, and Rialo's low fees make small-stake participation viable in a way that is not possible on Ethereum.
